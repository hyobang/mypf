COLOR TREE

다채로운 색의 나무가 자라나다

Overview

자기자신의 장단점과 특징을 파악하고 활용하기 위해 아이덴티티 분석 및 디자인을 진행하게 되었습니다.
해당 프로젝트를 통해서 웹디자이너로서의 잠재력을 표현하고자 합니다.

Goal

본인만의 아이덴티티를 명확히 분석하고 표현할 수 있습니다.
로고의 메타포를 통해 자신을 어떤식으로 표현하는지 확실히 전달할 수 있게합니다.

Self-Analysis

아이덴디티 디자인에 앞서 ‘나’에 대해 탐구해보았습니다.
마인드맵을 통해 그에 맞는 이미지와 키워드를 도출해 낼 수 있었습니다.

Slogan

COLOR TREE
다채로운 색의 나무가 자라나다

저만의 능력을 100% 활용하여 스스로를 빛내면서,
다채로운 색을 지니며 무한히 성장해 나아가는 디자이너가 되겠습니다.

Logo

처음 땅에서 씨를 뿌려 자라나는 새싹과, 하늘높이 날개를 펼쳐 날아가는 나비,
우리 생활에 밀접하게 연관되어 있는 꼭 필요한 나무 이 3가지를 조합하였습니다.
성장 + 긍정 + 유연 + 다양성을 결합하여 탄생된 저의 로고 디자인 입니다.

Font System

나무가 흙에서 자라나는 것처럼 또렷하면서도 거친 느낌의 폰트를 사용하고자 하였습니다.
강조되는 폰트에는 제주돌담체를 사용하였고,
서브폰트는 좀 더 심플하고 깔끔한 느낌의 나눔스퀘어네오체를 사용하였습니다.

Color System

가운데 기둥 부분을 나무의 몸통에 맞는 색상을 지정하고,
날개 위쪽에는 새싹의 의미인 연두색상을,아래쪽 날개에는나비의 밝은 노란색을 사용하여
3가지 심볼이 조화롭게 어울리도록 색상을 배치했습니다.
